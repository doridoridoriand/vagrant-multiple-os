# Multiple OS with Vagrant
## Dependencies
- vagrant (>= 2.2.19)
- Virtualbox (>= 6.1)
- vagrant-hosts

## How to use
```bash
$ vagrant plugin install vagrant-hosts
$ git clone git@github.com:doridoridoriand/vagrant-multiple-os.git
$ cd vagrant-multiple-os
$ cp configure.yml.sample configure.yml
$ vagrant up
```
